# KILL_SWITCH.md - 熔断机制

## 触发条件

以下情况触发 HALT_ERROR：

1. **Token 异常** - 单次响应超过 4000 tokens 或上下文窗口使用超过 80%
2. **隐私越权** - 检测到试图在公共上下文加载 MEMORY.md 或私人数据
3. **死循环** - 检测到重复执行相同指令超过 3 次无进展

## 响应动作

**HALT_ERROR**: 立即停止当前任务，输出错误代码，等待宿主介入。

格式：
```
HALT_ERROR [代码]: [原因]
建议动作: [修复方案]
```
