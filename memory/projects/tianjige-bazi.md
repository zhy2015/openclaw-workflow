# 天机阁 (bazi-fortune)

**项目状态**: ✅ 已上线运行
**部署地址**: http://124.222.70.196:30000
**技术栈**: Node.js + Express + SQLite + 阿里云百炼(kimi-k2.5)

## 功能模块

### 1. 八字算命
- 输入出生年月日时，自动排盘
- AI深度分析命理（2000-3500字）
- 命格大字展示
- 预估时间：2-3分钟

### 2. 手相分析
- 上传手相照片
- AI识别掌纹（智慧线/生命线/感情线/事业线/财运线）
- 分段卡片展示结果
- 预估时间：1-2分钟

### 3. 面相分析
- 上传面相照片
- AI识别五官（额头/眉毛/眼睛/鼻子/嘴巴/下巴）
- 分段卡片展示结果
- 预估时间：1-2分钟

## 技术架构

| 组件 | 说明 |
|------|------|
| 前端 | 纯HTML/CSS/JS，无框架依赖 |
| 后端 | Express + Multer（文件上传） |
| 数据库 | SQLite（存储历史记录） |
| AI模型 | 阿里云百炼 kimi-k2.5 |
| 熔断机制 | 每10分钟最多10次调用 |
| 进程管理 | PM2 |
