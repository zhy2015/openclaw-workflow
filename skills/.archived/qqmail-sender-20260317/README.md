# Email Sender 技能胶囊

## 发布指南

### 前置要求
- 安装 Node.js 和 npm
- 安装 ClawHub CLI: `npm install -g clawhub`

### 发布命令
```bash
clawhub login
cd skills/email-sender
clawhub publish --name "Email Sender" --tags "email,smtp,automation"
```

### 技能配置
- **名称**: Email Sender
- **标签**: email, smtp, automation, python
- **描述**: 通过Python自动发送邮件，支持附件和抄送
